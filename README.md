# Internship Portal

A full-stack Internship Portal developed using Spring Boot, React.js, and MySQL.

## Features

- Student Registration & Login
- Company Registration & Login
- Internship Posting
- Internship Applications
- Admin Dashboard
- Notifications
- Contact Module

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot
- Java
- REST API

### Database
- MySQL

## Installation

### Backend

cd backend
mvn spring-boot:run

### Frontend

cd frontend
npm install
npm start
